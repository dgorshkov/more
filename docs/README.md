# Documentation

This directory contains all project documentation organized by category.

## Structure

- **narrative/** - Story content, writing guidelines, and narrative design documents
- **design/** - Game design documents, mechanics, and gameplay specifications
- **technical/** - Technical documentation, architecture decisions, and API references

## Guidelines

- Documentation is kept up to date with implementation
- All documentation files use Markdown
- Diagrams are included where helpful (using Mermaid or other markdown-compatible formats)
- Related documents are linked together for easy navigation
