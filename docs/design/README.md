# Game Design Documentation

This directory contains all game design documents, mechanics specifications, and gameplay systems documentation.

## Contents

- **game-concept.md** - Core game concept, pillars, and design philosophy

## Purpose

Design documentation defines how the game plays, feels, and behaves. This includes:
- Core mechanics and systems
- Balance and tuning specifications
- Player experience goals
- UI/UX design
- Difficulty curves and progression
