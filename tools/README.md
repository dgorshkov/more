# Tools

This directory contains development tools, scripts, and utilities.

## Purpose

Store custom tools and scripts that aid development:
- Build scripts and automation
- Asset processing pipelines
- Development utilities
- Editor extensions or plugins
- Testing helpers
- Deployment scripts

## Guidelines

- Each tool should document its purpose and usage
- Tools should remain maintainable and well-commented
- Cross-platform compatibility should be considered
- Tool dependencies should be tracked in version control
